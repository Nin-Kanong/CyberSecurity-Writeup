## What is ZPhisher?
Zphisher is an open-source automated phishing toolkit written in Bash.


### Key Features
1. Multiple Templates – Includes cloned login pages of many popular sites.
2. Local Web Server – Uses PHP to host phishing pages on your machine.
3. Optional Tunneling – Can expose your local page temporarily through services like Cloudflare Tunnel or Localhost.run (for testing only).
4. Credential Logging (Lab Only) – Captures test credentials entered on the fake pages, stored locally in files.
5. Redirection – After submitting data, a user is redirected to the real website to make the simulation realistic.

## How ZPhisher Works
````
Zphisher Start
     ↓
Choose a website (Ex: Facebook)
     ↓
Zphisher downloads template
     ↓
Zphisher starts local PHP server
     ↓
Zphisher starts Ngrok/Cloudflared
     ↓
You get a phishing link
     ↓
Victim opens link (lab testing)
     ↓
Logs IP + Credentials
````
