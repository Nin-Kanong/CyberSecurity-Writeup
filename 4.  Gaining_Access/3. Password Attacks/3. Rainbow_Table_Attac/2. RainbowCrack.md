# RainbowCrack For Rainbow Attack


## What is RainbowCrack?
RainbowCrack is a password cracking tool that uses rainbow tables to crack password hashes significantly faster than traditional brute-force methods. It implements Philippe Oechslin's faster time-memory trade-off technique.

## Features
  - Rainbow table generation, sort, merge, conversion and lookup
  - Rainbow table of LM, NTLM, MD5, SHA1, SHA256 and customizable hash algorithms
  - Rainbow table of customizable charset
  - GPU acceleration with AMD GPUs (OpenCL technology)
  - GPU acceleration with NVIDIA GPUs (CUDA technology)
  - GPU acceleration with multiple GPUs
  - Command line and graphics user interface
  - Windows and Linux

#### General Command Format
| Item                  | Description                                         |
| --------------------- | --------------------------------------------------- |
| **rtgen**             | Rainbow table generator                             |
| **hash_algorithm**    | Hash type (lm, ntlm, md5, sha1, sha256)             |
| **charset**           | Character set (loweralpha, numeric, mixalpha, etc.) |
| **plaintext_len_min** | Minimum password length                             |
| **plaintext_len_max** | Maximum password length                             |
| **table_index**       | Table ID (0–n) to generate multiple tables          |
| **chain_len**         | Length of each chain                                |
| **chain_num**         | Number of chains                                    |
| **part_index**        | Part index (0–n) when splitting work                |


#### Hash Algorithms Table
| Algorithm  | Hash Length | Supported Password Length |
| ---------- | ----------- | ------------------------- |
| **lm**     | 8 bytes     | 0–7 characters            |
| **ntlm**   | 16 bytes    | 0–15 characters           |
| **md5**    | 16 bytes    | 0–15 characters           |
| **sha1**   | 20 bytes    | 0–20 characters           |
| **sha256** | 32 bytes    | 0–20 characters           |


#### Charset Options Table
| Charset Name             | Description                    |
| ------------------------ | ------------------------------ |
| **loweralpha**           | `abcdefghijklmnopqrstuvwxyz`   |
| **upperalpha**           | `ABCDEFGHIJKLMNOPQRSTUVWXYZ`   |
| **numeric**              | `0123456789`                   |
| **alpha**                | Upper + lower letters          |
| **alphanum**             | Letters + numbers              |
| **mixalpha**             | Lowercase + uppercase          |
| **mixalphanum**          | Lowercase + uppercase + digits |
| **ascii-32-95**          | Printable ASCII                |
| **custom:**`charset.txt` | Custom charset from a file     |

#### rtgen Options Table
| Option / Field        | Meaning                                                  |
| --------------------- | -------------------------------------------------------- |
| **hash_algorithm**    | Algorithm to generate rainbow for (md5, sha1, ntlm, lm…) |
| **charset**           | Defined charset (loweralpha, numeric, etc.)              |
| **plaintext_len_min** | Minimum password length to generate                      |
| **plaintext_len_max** | Maximum password length                                  |
| **table_index**       | Identifier for parallel/multiple tables                  |
| **chain_len**         | Chain length → affects speed and coverage                |
| **chain_num**         | Number of chains in table                                |
| **part_index**        | Split table generation (for big tables)                  |
| **-bench**            | Benchmark table performance                              |



### Components
RainbowCrack consists of three main tools:
  - rtgen - Rainbow table generator
  - rtsort - Sorts rainbow tables for faster lookup
  - rcrack - Cracks hashes using rainbow tables

````
sudo rtgen md5 loweralpha 1 5 0 1000 5000 0
````


- `md5`: We're cracking MD5 hashes
- `loweralpha`: Passwords contain only lowercase letters (a-z)
- `1 5`: Passwords are 1 to 5 characters long
- `0`: Table version 0
- `1000`: How complex the table is
- `5000`: How many passwords to include
- `0`: Part 0 (first part)
