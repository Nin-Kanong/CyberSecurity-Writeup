## What is John the Ripper?
John the Ripper (often called John) is a fast password cracker, currently available for many flavors of Unix, Windows, DOS, and OpenVMS. Its primary purpose is to detect weak Unix passwords.


## Key Features:
  - Multiple password hash types - 300+ hash formats supported
  - Multiple attack modes - Wordlist, single crack, incremental, external
  - Highly optimized - Written in C, uses SIMD and multiple cores
  - Cross-platform - Linux, Windows, macOS, BSD
  - Extensible - Custom rules, formats, and modes

## John vs Other Tools:
  - John: Best for offline password cracking, hash cracking, local passwords
  - Hydra/Medusa: Network login brute-forcing
  - Hashcat: GPU-optimized, often faster for complex hashes
