## What is Kerbrute?

Kerbrute is a specialized tool for password spraying attacks against Kerberos authentication, specifically targeting Active Directory environments. It's designed to be fast and avoid account lockouts.

## Why Kerbrute is Special:
- Built specifically for Kerberos/Active Directory
- Bypasses many traditional detection methods
- Very fast compared to other tools
- Written in Go (single binary, no dependencies)
