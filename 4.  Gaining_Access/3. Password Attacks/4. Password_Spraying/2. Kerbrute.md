## What is Kerbrute?
**Kerbrute** is a specialized tool for password spraying attacks against Kerberos authentication, specifically targeting Active Directory environments. It's designed to be fast and avoid account lockouts.

## Why Kerbrute is Special:
- Built specifically for Kerberos/Active Directory
- Bypasses many traditional detection methods
- Very fast compared to other tools
- Written in Go (single binary, no dependencies)

## Kerbrute is designed for Kerberos authentication in Active Directory environments.

### Available Commands Table
| Command           | Description                                                        |
| ----------------- | ------------------------------------------------------------------ |
| **bruteforce**    | Bruteforce `username:password` combos (from file or stdin).        |
| **bruteuser**     | Bruteforce a **single userâ€™s password** using a wordlist.          |
| **passwordspray** | Test **one password** against a list of users (password spraying). |
| **userenum**      | Enumerate valid AD usernames via Kerberos.                         |
| **version**       | Show Kerbrute version information.                                 |
| **help**          | Show help for commands.                                            |



### Global Flags Table
| Flag | Long Flag   | Description                                               | Example                |
| ---- | ----------- | --------------------------------------------------------- | ---------------------- |
| -d   | `--domain`  | Domain name (FQDN) to target.                             | `--domain contoso.com` |
|      | `--dc`      | Domain Controller IP/hostname. If blank, resolve via DNS. | `--dc 192.168.1.10`    |
|      | `--delay`   | Delay (ms) between attempts. Forces single thread.        | `--delay 200`          |
| -o   | `--output`  | Save output logs to file.                                 | `--output results.txt` |
|      | `--safe`    | Stop if any user becomes locked out.                      | `--safe`               |
| -t   | `--threads` | Number of threads (default 10).                           | `--threads 20`         |
| -v   | `--verbose` | Show all failures and errors.                             | `--verbose`            |
| -h   | `--help`    | Show help message.                                        | `kerbrute -h`          |



### Usage Format Table
| Usage                       | Meaning                                                              |
| --------------------------- | -------------------------------------------------------------------- |
| `kerbrute [command]`        | Run Kerbrute with any available command.                             |
| `kerbrute [command] --help` | Show help for a specific command (e.g., `kerbrute userenum --help`). |
