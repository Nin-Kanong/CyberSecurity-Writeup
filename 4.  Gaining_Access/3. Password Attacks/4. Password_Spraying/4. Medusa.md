## What is Medusa?

Medusa is a command-line tool for network login brute-forcing (password spraying). It's designed to be:
 - Parallel: Can run multiple attempts simultaneously
 - Modular: Supports many protocols via plugins
 - Flexible: Works with various service types
 - Lightweight: No GUI, fast execution


## SSH Password Spraying

### With single user
````
medusa -h 192.168.127.129 -U /usr/share/wordlists/user.txt -p msfadmin -M ssh -t 5
````
