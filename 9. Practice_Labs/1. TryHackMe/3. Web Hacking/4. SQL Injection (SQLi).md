# Task 1: Brief

## What is SQL?
**SQL (Structured Query Language)**: is a programming language used to manage and interact with relational databases (like MySQL, PostgreSQL, SQLite, Microsoft SQL Server, etc.). Web applications often use SQL to:
- Store user data (e.g., usernames, passwords, emails)
- Retrieve information (e.g., product listings, user profiles)
- Update or delete records

## What Going on
A **Database** is like a big filing cabinet that stores usernames, passwords, and customer info.
**SQL** is the language websites use to talk to that filing cabinet.
**SQL Injection (SQLi)** happens when a hacker sneaks bad instructions into that conversation.
