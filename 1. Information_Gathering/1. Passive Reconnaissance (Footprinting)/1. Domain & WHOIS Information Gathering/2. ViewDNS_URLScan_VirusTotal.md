# 2. URLScan

### What is URLScan?
URLScan is an online sandbox/webpage scanner that fetches and analyzes a URL, shows requests, response headers, loaded resources, external connections, page screenshot and more, great for investigating suspicious URLs and web app behavior.

- Go to browser: https://urlscan.io/

### Option usage of URLScan:
| Option / Feature       | Input                       | Example                                    | Purpose                                  |
| ---------------------- | --------------------------- | ------------------------------------------ | ---------------------------------------- |
| Scan URL               | URL                         | [https://example.com](https://example.com) | Submit a website to analyze its behavior |
| Visibility             | Public / Unlisted / Private | Public                                     | Control who can see the scan results     |
| Country                | Country code                | US, DE                                     | Choose scan location                     |
| User Agent             | Browser type                | Chrome / Firefox                           | Simulate different browsers              |
| Referer                | URL                         | [https://google.com](https://google.com)   | Set HTTP referer header                  |
| Tags                   | Custom text                 | phishing, malware                          | Organize and label scans                 |
| Screenshot             | Enable / Disable            | Enabled                                    | Capture webpage screenshot               |
| DOM Snapshot           | Enable                      | On                                         | Analyze page structure & scripts         |
| JavaScript Execution   | Enable / Disable            | Enabled                                    | See JS behavior                          |
| Redirect Tracking      | Auto                        | Enabled                                    | Follow HTTP redirects                    |
| DNS Requests           | Auto capture                | —                                          | Show DNS lookups made                    |
| HTTP Requests          | Auto capture                | —                                          | View all network requests                |
| TLS / SSL Info         | Auto capture                | —                                          | Inspect certificate & encryption         |
| Technologies Detection | Auto                        | —                                          | Detect CMS, frameworks, libraries        |
| WHOIS Lookup           | Auto                        | —                                          | Domain ownership info                    |
| IP Resolution          | Auto                        | —                                          | Resolve domain → IP address              |
| Hashes                 | Auto                        | —                                          | File and resource hashes                 |
| Malware Indicators     | Auto                        | —                                          | Detect suspicious behavior               |
| Search Scans           | Keyword                     | domain:example.com                         | Find previous scans                      |
| API Access             | API key                     | —                                          | Automate scanning & search               |
| Download Report        | PDF / JSON                  | —                                          | Export scan results                      |


### Common Use Cases
| Use Case          | Why URLScan Helps                      |
| ----------------- | -------------------------------------- |

| Malware research  | Detect malicious JS & downloads        |

# 3. VirusTotal
### What is VirusTotal?
VirusTotal is an aggregated malware/IOC analysis service — analyze files, URLs, IPs and domains against many antivirus engines and threat feeds. Also shows historical behavior, community comments, and relationship graphs.

- https://www.virustotal.com/gui/home/upload
