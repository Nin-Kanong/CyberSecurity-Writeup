- `-A`: Enables OS detection, version detection, script scanning, and traceroute.
- `-v`: Verbose output: shows more scan progress and details.
- **This command performs a deep scan across your local subnet, trying to:
  ‚Ä¢ 	Detect live hosts

- Why It's Useful:
  - Recon in exam labs

  - Documenting host/service changes with `ndiff`
  - Building GitHub recon notes

- Connected nodes: These are the IPs discovered during the scan (e.g., 192.168.127.1, .2, .128, .254).
- Lines between nodes: Show network relationships or scan paths (like traceroute hops).
