### 
````
ls -l /usr/share/nmap/scripts | grep ssh
````
<img width="605" height="168" alt="image" src="https://github.com/user-attachments/assets/473031fb-c1a2-470a-afbf-4c58a03ff39d" />


- `ls -l`: Lists files in long format (permissions, owner, size, date)
- `/usr/share/nmap/scripts`: Directory containing all Nmap NSE scripts.
- `| grep ssh`: Filters output to show only lines containing “ssh”, i.e., SSH-related scripts
- **This is useful for:
  • 	Identifying available SSH-related NSE scripts
  • 	Planning targeted scans like brute-force, host key fingerprinting, or version checks**
